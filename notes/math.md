# Math

```math
`    = backtick
=    = equals sign
≠    = not equals sign
+    = plus sign
-    = minus sign
×    = times sign
÷    = division sign / obelus
&    = ampersand
-     = hyphen
:     = colon
( )   = round brackets
{ }   = curly brackets or braces
[ ]   = box brackets
< >   = angle brackets
#     = hatch
,     = comma
;     = semicolon
/     = slash
\     = backslash
~    = tilde
_    = underscore
|  = Vertical bar
```
